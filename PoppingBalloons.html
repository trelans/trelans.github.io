Popping Balloons
  An ASMR mobile hyper casual game created by Unity and tested by CrazyLabs. You have a needle and there are a lot of balloons. Escape from obstacles and pop all the balloons!
Project Date
July 2021
The Challenge
  This was my first 3D Game and also first hyper casual game. Implementing a publisher's SDK was also new for me for this project.
Project Role
This was an indie game so I had done 2D Design, 3D Design, Game Design, Development, Video Creation
My Approach
I inspired from stack ball. I used fever mode and cylinder.
